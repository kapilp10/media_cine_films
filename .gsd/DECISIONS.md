# DECISIONS.md

## Title: Framework Selection
**Date:** 2026-02-26
**Context:** Need a stack for a high-performance, aesthetically pleasing portfolio website.
**Decision:** Next.js (App Router) + Tailwind CSS
**Rationale:** Standard for modern React applications, excellent performance and SEO, and Tailwind CSS provides the utility-first approach needed to rapidly match the specific "Nakula" aesthetic.
